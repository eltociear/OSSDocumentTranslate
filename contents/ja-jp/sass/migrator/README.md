Sass移行機能は、非推奨の動作を使用するスタイルシートを最新かつ最高のSass機能に自動的に更新します。
npmや他の場所から `sass-migrator`として簡単にインストールできます。

詳細およびインストール手順については、[SassのWebサイト][]を参照してください。

[SassのWebサイト]: https://sass-lang.com/documentation/cli/migrator
